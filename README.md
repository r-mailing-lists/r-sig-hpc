# r-sig-hpc

High-Performance Computing

Automated mbox archive. Updated daily via GitHub Actions.

## Structure

- `raw/` - Monthly .mbox files downloaded from [https://stat.ethz.ch/pipermail/r-sig-hpc/](https://stat.ethz.ch/pipermail/r-sig-hpc/)
- `processed/` - Structured JSON output from rmail-parser
- `meta.json` - List statistics
